# FastPick 全局图标优化计划

## 项目概述
将 FastPick 应用的 UI 从文字按钮优化为图标按钮，提升视觉简洁性和国际化友好性。

---

## 一、图标选型

使用 **Segoe Fluent Icons** 字体（Windows 11 原生图标字体），无需额外引入图标库。

---

## 四、技术实现要点

### 4.1 图标按钮样式
```xml
<Style x:Key="IconButtonStyle" TargetType="Button">
    <Setter Property="FontFamily" Value="Segoe Fluent Icons"/>
    <Setter Property="FontSize" Value="16"/>
    <Setter Property="Width" Value="40"/>
    <Setter Property="Height" Value="32"/>
    <Setter Property="Background" Value="Transparent"/>
    <Setter Property="Foreground" Value="{StaticResource TextPrimaryBrush}"/>
</Style>
```

### 4.2 带文字图标按钮样式
```xml
<Style x:Key="IconTextButtonStyle" TargetType="Button">
    <Setter Property="FontFamily" Value="Segoe Fluent Icons, Segoe UI"/>
    <Setter Property="FontSize" Value="14"/>
    <Setter Property="Padding" Value="12,8"/>
</Style>
```

### 4.3 ToolTip 样式
```xml
<ToolTipService.ToolTip>
    <ToolTip Content="功能说明"/>
</ToolTipService.ToolTip>
```
