Windows 看图选片应用 UI 设计文档（基于草图修正版）
一、文档说明
本文档严格对齐你提供的六张界面草图，基于 Fluent UI + WinUI 3 设计规范，以 Markdown 可视化形式呈现应用的四段式界面结构、组件布局及交互逻辑，作为开发和验收的 UI 标准依据。
二、整体界面概览（四段式）
plaintext
┌─────────────────────────────────────────────────────────────────────────────┐
│ [路径] [导出] [删除] [筛选] [设置] │ 标题栏（带窗口控制：最小化/最大化/关闭） │ 标题栏
├─────────────────────────────────────────────────────────────────────────────┤
│ [抽屉式展开区]                                                                 │
│ （与顶部按钮联动，如路径/导出/删除/筛选/设置的子选项）                          │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│                                   [预览区]                                   │
│                            （单图/最多5张平铺展示）                          │ 预览区
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│ [缩略图1] [缩略图2] [缩略图3] ... （横向滚动，网格布局）                      │ 缩略图区
├─────────────────────────────────────────────────────────────────────────────┤
│ [文件信息抽屉]                          │ 缩放控制（60%/滚轮/放大镜） │            │ 底栏
└─────────────────────────────────────────────────────────────────────────────┘
整体为四段式垂直布局，新增了顶部抽屉式展开区，与标题栏按钮联动。
所有组件遵循 Fluent UI 设计语言，圆角、阴影、交互反馈符合 Windows 原生体验。
界面响应式：窗口缩放时，预览区自动适配，缩略图区保持横向滚动，底部工具栏固定高度。
三、分区域详细设计
3.1 标题栏（CommandBar 组件）
布局结构
plaintext
┌───────────────────────────────────────────────────────────────────────────────────────────────┐
│ [路径] [导出] [删除] [筛选] [设置] │ 标题栏 │ ? │ 最小化 │ 最大化 │ 关闭 │
└───────────────────────────────────────────────────────────────────────────────────────────────┘
组件说明
表格
组件	交互逻辑	视觉样式
路径按钮	点击展开路径抽屉，显示双路径选择（路径 1 / 路径 2）+ 选择 / 载入按钮	Fluent 圆角按钮，hover 时高亮背景
导出按钮	点击展开导出抽屉，显示 RAW 导出控制、文件夹配置、导出路径 + 导出按钮	主色调按钮（建议 #0078D4）
删除按钮	点击展开删除抽屉，显示当前预删除张数、删除选项（同时删除 / 仅 JPG / 仅 RAW）	文字和图标为红色，hover 时加深
筛选按钮	点击展开筛选抽屉，显示星级筛选（有星 / 无星 / 特定星）、文件类型筛选（双格式 / 仅 JPG / 仅 RAW）	选中筛选条件后按钮高亮（蓝色边框）
设置按钮	点击展开设置抽屉，显示常见参数配置项	普通按钮样式
窗口控制	最小化 / 最大化 / 关闭按钮，位于最右侧	标准 Windows 窗口控制样式
交互细节
点击任意功能按钮（路径 / 导出 / 删除 / 筛选 / 设置），在标题栏下方展开对应的抽屉式子选项区，再次点击或点击其他按钮切换抽屉。
抽屉展开时，标题栏按钮保持高亮，指示当前激活的功能区。
3.2 抽屉式展开区（与按钮联动）
路径抽屉（点击 “路径” 按钮展开）
plaintext
┌─────────────────────────────────────────────────────────────────────────────┐
│ 路径1：[输入框] [选择]  路径2：[输入框] [选择] [载入]                         │
└─────────────────────────────────────────────────────────────────────────────┘
功能：双路径选择，支持手动输入路径或通过 “选择” 按钮打开文件夹选择器，“载入” 按钮触发图片扫描加载。
异常处理：路径失效时，点击 “载入” 自动提示重新选择。
导出抽屉（点击 “导出” 按钮展开）
plaintext
┌─────────────────────────────────────────────────────────────────────────────┐
│ raw文件导出控制：[下拉框：5星以上]  导出文件夹名配置：jpg[默认：JPG] RAW[默认：RAW]    │
│ 路径导出位置：[输入框]         [导出]                                           │
└─────────────────────────────────────────────────────────────────────────────┘
功能：默认导出带星的jpg，控制 RAW 导出的评级阈值（默认 5 星以上），选择导出的RAW文件评级的阈值（1-5星），自定义导出文件夹名称和路径。
导出后自动创建 JPG/RAW 子文件夹。
删除抽屉（点击 “删除” 按钮展开）
plaintext
┌─────────────────────────────────────────────────────────────────────────────┐
│ 当前预删除：[x张]  [jpg、raw同时删除] [仅删除jpg] [仅删除raw]              │
└─────────────────────────────────────────────────────────────────────────────┘
功能：显示当前预删除的图片数量，提供删除选项，点击后弹出确认弹窗。[x张]是个按钮，点击可实现筛选视图，筛选出当前预删除的图片。
预删除数量实时更新，按钮文本为红色。
筛选抽屉（点击 “筛选” 按钮展开）
plaintext
┌─────────────────────────────────────────────────────────────────────────────┐
│ 全部[x张]  双文件[x张]  仅jpg[x张]  仅raw[x张]                     │
│ 有星/无星/等于/大于等于/小于等于/★★★★★                                  │
└─────────────────────────────────────────────────────────────────────────────┘
功能：按文件类型筛选（双格式 / 仅 JPG / 仅 RAW）和星级筛选（有星 / 无星 / 等于 /≥/≤ + 0-5 星）组合筛选。
筛选后自动更新缩略图区和预览区，筛选按钮高亮当前条件。
设置抽屉（点击 “设置” 按钮展开）
plaintext
┌─────────────────────────────────────────────────────────────────────────────┐
│                              设置界面 - 常见参数配置                          │
│                                                                             │
│                              [保存] [取消]                                  │
└─────────────────────────────────────────────────────────────────────────────┘
功能：预留常见设置项（如默认导出路径、默认评级阈值、RAW 解码优先级等），底部有保存 / 取消按钮。

布局结构（单图模式）

布局结构（多选模式，最多 5 张）
plaintext
┌─────────────────────────────────────────────────────────────────────────────┐
│ [🖼️1]  [🖼️2]  [🖼️3]  [🖼️4]  [🖼️5]                                         │
│ （平铺展示，宽120:高80，间距5px）                                              │
└─────────────────────────────────────────────────────────────────────────────┘
组件说明
表格
组件	交互逻辑	视觉样式
单图预览 Image	支持鼠标滚轮缩放、鼠标拖动平移；双击还原 100% 比例	居中显示，Stretch=Uniform（保持比例）
多图平铺 ItemsRepeater	最多显示 5 张，超过 5 张不展示；每张图带灰色边框（1px）	网格布局（5 列 1 行），每张图最小尺寸 200×150px
加载进度环	图片加载中显示，居中覆盖在预览区	白色半透明背景，尺寸 50×50px
交互细节
点击缩略图切换单图预览，按住 Ctrl 点击缩略图进入多选模式。
多选模式下，预览区自动切换为平铺布局，取消多选后切回单图。
缩放控制：底部工具栏的数字框可调整缩放比例（50%-200%），实时同步预览区。

┌─────────────────┐
│ 🔴 [🖼️缩略图]   │
│ ★★★☆☆    [JPG] │

组件说明
表格
组件	交互逻辑	视觉样式
缩略图 Image	点击选中，Ctrl + 点击多选，Shift + 点击连续选择	尺寸 100×80px，Stretch=UniformToFill
选中边框	选中时显示蓝色边框（2px），未选中时透明	圆角 2px，高亮显示选中状态
预删除角标 Ellipse	标记为预删除时显示，位于右上角	红色实心圆，尺寸 15×15px
星级 RatingControl	点击可修改星级（0-5），支持批量修改	位于左下角，尺寸 60×20px，半透明背景
文件类型标签 TextBlock	显示文件类型（JPG/RAW/ 双格式）	黑色背景白色文字，字体大小 8px，位于右下角
交互细节
同文件名的 JPG 和 RAW 自动折叠为一个缩略项，文件类型标签显示 “双格式”。
鼠标悬停缩略项时，显示完整文件名提示框。
滚动时懒加载：仅加载可视区域的缩略图，滚动后动态加载新区域。

表格

文件信息抽屉展开后，显示当前选中图片的详细信息，支持一键在资源管理器中打开文件路径。
缩放控制支持鼠标滚轮调整，实时同步预览区。

五、交互规范
5.1 快捷键列表
表格
快捷键	功能
0-5	为选中图片设置对应星级（0 星为取消评级）
Delete	标记 / 取消标记选中图片为预删除
Ctrl+A	全选所有图片
Shift + 点击	连续选择缩略图
Ctrl + 滚轮	调整预览区缩放比例
←/→	切换上一张 / 下一张图片
Esc	关闭抽屉 / 弹窗，取消多选
5.2 视觉反馈规范
操作成功：短暂显示 Toast 提示（如 “已标记 5 张图片为预删除”）。
操作失败：弹出错误提示框（如 “路径无效，请重新选择”）。
加载状态：按钮 / 区域显示加载动画，禁止重复操作。
选中状态：所有选中元素统一蓝色高亮（#0078D4）。
六、适配规范
6.1 分辨率适配
最小窗口尺寸：1280×720px。
最大窗口尺寸：支持全屏，布局自动拉伸。
缩略图区：窗口宽度变化时，自动调整列数（最小 4 列，最大 12 列）。
6.2 系统适配
支持 Windows 10 1903+ / Windows 11。
跟随系统主题（浅色 / 深色模式），组件自动适配。
高 DPI 适配：支持 100%/125%/150%/200% 缩放比例。

七、总结
应用 UI 严格遵循草图中的五段式结构（标题栏 - 抽屉区 - 预览区 - 缩略图区 - 底部工具栏），新增了抽屉式交互，完全匹配你的设计意图。
核心交互（路径加载、导出、删除、筛选）均通过抽屉式展开实现，保持界面简洁高效，所有功能入口清晰可见。
界面适配多分辨率和系统主题，缩略图懒加载保证 2000 + 图片加载流畅，符合 “简洁高效” 的核心设计目标。
可基于此 UI 文档直接开展前端开发，所有组件和交互逻辑均已明确，后续只需绑定业务逻辑即可完成完整应用开发。
